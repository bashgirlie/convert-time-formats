# convert-time-formats
Convert to or from Slash and ISO datetime formats



### Usage

python3 convert_time.py "06/11/2026 9:55:52"

- To print the current time in ISO 8601 format after converting the timestamp
python3 convert_time.py "06/11/2026 9:55:52" --current-time

- To..
python3 convert_time.py 20260611165448.285

python3 convert_time.py 20260611165448.285 --current-time

python3 convert_time.py --current-time
